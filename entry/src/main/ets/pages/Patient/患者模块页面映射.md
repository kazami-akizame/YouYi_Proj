## 统一预览入口

- 预览容器：`entry/src/main/ets/pages/PatientModulePreview.ets`
- 应用入口页：`entry/src/main/ets/pages/Index.ets`
- 使用方式：运行应用后，在左侧菜单点击页面名称，可查看右侧对应 UI 效果。

## 个人信息子流程（简要）

- **患者姓名 / 备注**：右侧 `u1158` → 路由「编辑姓名」「编辑备注」。
- **性别**：右侧 `u1158` → 底部弹层（标题「请选择患者性别」，选项「男」「女」，「取消」关闭），选中后更新列表展示。
- **年龄**：右侧 `u1158` → 路由「编辑个人年龄」`EditPersonalAgePage`（与预览中的 `patientAgeDisplay` 双向同步）。
